# SecureNet — Cybersecurity Chatbot
A WPF-based Cybersecurity Chatbot built in C# for PROG6221.

## Student
- **Name:** Musa Boltina
- **Student Number:** ST10471931

## Features
### Part 1
- Voice greeting on startup
- ASCII art display
- User name capture and personalised responses
- Keyword-based cybersecurity responses
- Sentiment detection
- Fallback responses

### Part 2
- WPF GUI with dark GitHub-style theme
- Chat interface with message bubbles
- 10 topic buttons in sidebar
- Follow-up handling (tell me more)
- Persistent user memory saved to file

### Part 3
- Task Assistant with JSON storage (add, view, complete, delete)
- 12-question Cybersecurity Quiz with scoring and feedback
- NLP intent detection from varied phrases
- Activity Log with timestamps and show more option

## Prerequisites
- Visual Studio 2022
- .NET 8.0
- Newtonsoft.Json (install via NuGet: right-click project → Manage NuGet Packages → search Newtonsoft.Json → Install)

## How to Run
1. Clone the repo: `git clone https://github.com/Musa334-cloud/ST10471931_Musa_Boltina_POE.git`
2. Open `ST10471931_Musa_Boltina_POE.slnx` in Visual Studio 2022
3. Install Newtonsoft.Json via NuGet if not already installed
4. Press F5 to run

## Notes
- `greeting.wav` is included and set to Copy Always
- `tasks.json` is auto-created when the first task is added
